Font

Google Fonts is an adequate free resource for fonts. Through a link in the head or an import statement in a stylesheet, the fonts are loaded from Google servers when your page loads.

For a wider variety, you can find free fonts at DaFont, fontspace, or FontSquirrel. Unlike Google Fonts, from these sites, you will download font files, which will need to be included as assets on your web server and imported with a stylesheet. Note that not all fonts are free or in the case of DaFont, public domain.

Usually, provided font files are of one or two types. You may need to use a font file converter, such as the aptly named font converter or the preferred transfonter, which can convert multiple font files into multiple formats.
